# dev-talent-ranker

## 一、项目概述

**项目简介：**DevTalent Ranker是一个基于GitHub开源项目数据的开发者能力评估平台。通过分析开发者在项目中的贡献度和项目的重要程度，我们为开发者提供技术能力的评级（TalentRank）。同时，平台能够推测开发者的所属国家（Nation）并支持按领域搜索匹配开发者，助力企业和社区发现和评价技术人才。
