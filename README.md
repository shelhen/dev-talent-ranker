#### （1）基础功能（必须实现）

- 开发者在技术能力方面 `TalentRank`（类似 `Google` 搜索的 `PageRank`），对开发者的技术能力进行评价/评级。评价/评级依据至少包含：项目的重要程度、该开发者在该项目中的贡献度。
